PASOS PARA EJECUTAR EL JUEGO CON NPM

1. Descomprime el ZIP.
2. Entra a la carpeta donde esta package.json.
3. Ejecuta:

   npm install
   npm run dev

4. Abre el enlace que aparece, normalmente:

   http://localhost:5173

IMPORTANTE:
- Ejecuta npm install SOLO en la carpeta que contiene package.json.
- Este ZIP ya no trae package-lock.json antiguo.
- Se corrigio package.json para usar dependencias estables:
  Phaser 3.90.0 + Vite 5.4.21.
